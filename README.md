# audio-experiments

A hands-on journey into digital audio — processing, synthesis, and manipulation — built from the ground up in C. The goal is to understand what's really happening at the level of samples, sample rates, and raw file formats.

Each numbered directory is a self-contained experiment that builds on the ideas from the last.

## Roadmap

Working from the fundamentals outward — future experiments will explore envelopes, additive and subtractive synthesis, filters, effects (delay, reverb), and eventually real-time processing.
